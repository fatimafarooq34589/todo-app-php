# 📝 Todo App (PHP + MySQL)

## 📌 Project Description
This is a simple Todo App built using PHP and MySQL.  
It performs full CRUD operations:

- Add new task
- View all tasks
- Update task (edit & mark complete/pending)
- Delete task

---

## ⚙️ Setup Instructions (XAMPP)

1. Install XAMPP
2. Move project folder to:
   C:\xampp\htdocs\
3. Start Apache & MySQL
4. Open browser and go to:
   http://localhost/todoApp/

---

## 🗄️ Database Setup

1. Open phpMyAdmin
2. Create database:
   todo_app
3. Import file:
   database.sql

---

## 🎯 Features

✔ Add Task  
✔ Edit Task  
✔ Delete Task  
✔ Mark Complete / Pending  
✔ Simple UI with styling
